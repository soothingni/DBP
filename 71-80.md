# 80. 믿음직함

공리1. 문장 X의 믿음직함 Pr(X)는 0과 1 사이의 숫자이다.

공리2. 참말이라는 것을 아는 문장의 믿음직함은 1이다.

공리3. 문장 X와 Y가 뜻이 같다면, Pr(X) = Pr(Y)

공리4. 문장 X와 Y가 일관되지 않다면, Pr(X이거나 Y) = Pr(X) + Pr(Y)



정리1. Pr(X는 거짓이다) = 1 - Pr(X)

정리2. 거짓말이라는 것을 아는 문장의 믿음직함은 0이다.

정리3. X로부터 Y가 반드시 따라 나온다는 것을 안다면, Pr(X) <= Pr(Y)

정리4. Pr(X이거나 Y) = Pr(X) + Pr(Y) - Pr(X이고 Y)



## 가.

1. X로부터 반드시 Y가 따라나온다면, Pr(X) = Pr(X이고 Y)

   X로부터 반드시 Y가 따라나온다면 X와 "X이고 Y"는 뜻이 같다. X가 참이면 Y도 반드시 참이기 때문에 모든 가능세계에서 X와  "X이고 Y"의 진리값이 같기 때문이다. 따라서 공리3에 의해 Pr(X) = Pr(X&Y). 

2. Pr(X) = Pr(X이고 Y) + Pr(X이지만, Y는 거짓이다)

   Pr(X) = Pr(X&Y) + Pr(X&~Y)

   "X&Y"와 "X&~Y"는 일관되지 않다. 따라서 공리4에 의해 우변을 다음과 같이 바꿀 수 있다. Pr(X&Y) + Pr(X&~Y) = Pr('X&Y'v'X&~Y') = Pr(X&'Yv~Y'). 'Yv~Y'는 반드시 참말이기 때문에 Pr(X&'Yv~Y')는 Pr(X)와 같다. 따라서 우변과 좌변이 일치한다.

3. Pr(X이거나 Y) = Pr(X이고 Y) + Pr(X이지만, Y는 거짓이다) + Pr(X는 거짓이고 Y)

   Pr(XvY) = Pr(X&Y) + Pr(X&~Y) + Pr(~X&Y)

   "X이거나 Y"에 거짓이다 넣어, Pr(XvY) = 1-Pr(~X&~Y). 

   Pr(X&Y) + Pr(X&~Y) + Pr(~X&Y)는 모든 가능 세계에서 Pr(~X&~Y)만을 배제하고 있으므로 우변 역시 1-Pr(~X&~Y)로 바꿀 수 있다. 따라서 좌변과 우변이 일치한다.
